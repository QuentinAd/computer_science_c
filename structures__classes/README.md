# Structures & Classes

This folder contrasts a C `struct` with a simple C++ `class`.

- `struct_example.c` demonstrates defining a `Person` structure and passing it to a function.
- `class_example.cpp` shows a `Rectangle` class with a method that computes its area.

Compile the C example:

```bash
clang struct_example.c -o struct_example
```

Compile the C++ example:

```bash
clang++ class_example.cpp -o class_example
```
