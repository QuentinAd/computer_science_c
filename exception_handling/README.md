# Exception Handling

This directory presents error handling approaches in C and C++.

- `error_codes.c` illustrates returning a boolean to report errors.
- `try_catch.cpp` demonstrates C++ exceptions using `try`/`catch`.

Compile the C example:

```bash
clang error_codes.c -o error_codes
```

Compile the C++ example:

```bash
clang++ try_catch.cpp -o try_catch
```
